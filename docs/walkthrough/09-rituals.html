Chapter 09

Rituals

A ritual gathers a few habits into a named flow you move through together — a morning routine — tap by tap. It's habit-stacking (a proven behavioural technique) turned into a calm, guided experience.

What a ritual is

Just a name and an ordered list of habit ids: { id, name, habitIds[] }. It doesn't own habits — it references them, so a habit can be in a ritual and still live normally on Home. lib/rituals.ts → ritualHabits() resolves the live habits in order, quietly skipping any that were deleted or archived, so a ritual never breaks when its contents change.

Decision · stored on the settings rowRituals are small per-account data, so instead of a whole new synced table (with its own migration, sync mappings, realtime, and outbox), they ride the existing notification_settings.rituals jsonb column. They sync across devices for free via the settings path. The trade-off — no per-ritual realtime granularity — is fine for something you build once and rarely edit.

Building one — RitualSheet

Name it, then tap tend-habits to add them in order. Each selected habit shows an order badge (1, 2, 3…) so the flow is legible at a glance. Editing and deleting share the same sheet.

Doing one — RitualPlayer

Tapping Begin opens a full-screen, distraction-free flow:

  • One habit at a time, big serif name, a watercolour wash in that habit's colour.
  • Progress dots across the top; the current step widens.
  • A large circle to complete — tapping it logs the habit (via the normal useLogs.toggle, so it counts everywhere) and eases to the next step.
  • Habits already done today start complete, so a mid-morning run just continues from where you are.
  • A "Skip for now" escape hatch, and a gentle "Ritual complete" finish.

On Home, each ritual card shows its step count and today's progress, with a Begin button that becomes Review once everything in it is done.

Why a full-screen playerThe delight of habit stacking is flow — doing the set as one calm sequence, not ticking a checklist. The player removes every other affordance so, for two minutes, there's just you and the next small thing. It logs through the same store as tapping on Home, so there's no separate data path to keep in sync.
